# Svelte + Vite + Router

This template should help get you started developing with Svelte quickly.

## Usage

```shell
git clone -b svt_router --depth=1 git@github.com:webkong/svelte-vite-template.git
cd svelte-vite-template
npm i
npm run dev
```
